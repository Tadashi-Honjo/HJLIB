＜概要＞
TSXBINはシンボル定義に元にバイナリデータを表示する事ができます。
データレベルの不具合などを解析するときなどに便利です。

＜使用方法＞
以下のサイトなどでTSXBINをダウンロードしてください。
http://www.vector.co.jp/soft/dl/win95/util/se094825.html

TSXBIN.exeがあるフォルダ内に解析したいシンボル定義ファイルを追加してください。
HJLIB独自フォーマットなどはTYPEDEF_HJ.Hの追加も必須です。
シンボル定義ファイル同様に追加してください。

TSXBINでバイナリを読み込むと
シンボル定義を元に表示してくれます。
